Документация yandex_oauth
=========================

   | *Спасибо, что обратились к нашей библиотеке!*

**yandex_oauth** - библиотека функций для работы с OAuth токенами Яндекс API

.. note::
   Библиотека находится в активной стадии разработки.

На данный момент реализованы функции получения токенов по коду авторизации, сохранения в и загрузку из pickle хранилища.
